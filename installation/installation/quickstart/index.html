    How can I help?

    MacOS and Linux users can help by adding and verifying information on how to install the dependencies on their respective operating systems.

    These are quickstart instructions to get you started immediately.

    Warning

    Only follow these instructions if you're already familiar with Python or installing packages yourself. If you're new, we suggest following the standard installation guide instead.

    Pre-installation

    It may be a good idea to get a clean slate before continuing.

    Removing existing installations

    When to do this

    If you're a programmer, you might want to keep your existing Python installation, but if you're not and you want to start fresh, this step is recommended.

    Prior to (re-)installing Vapoursynth, make sure to remove any existing installations.

    This means deleting the following directories:

    • %APPDATA%/VapourSynth
    • %APPDATA%/Python
    • %LOCALAPPDATA%/Programs/VapourSynth
    • %LOCALAPPDATA%/Programs/Python
    • ~/Library/Application Support/VapourSynth
    • ~/Library/Application Support/Python
    • ~/Library/Python
    • ~/.config/VapourSynth

    Installation

    Installing JET and its dependencies is pretty straightforward.

    Installation steps

    1. Install the latest version of Python from the Python website.
      Make sure to select the option to "Add Python to PATH"!

    2. Download the latest GitHub release for Vapoursynth.

    3. Install JET using pip:

    1
    $ pip install vsjet
    1. Install Python using Homebrew.
    1
    $ brew install python
    1. Install Vapoursynth using Homebrew.
    1
    $ brew install vapoursynth
    1. Install JET using pip:
    1
    $ pip install vsjet

    1. Add the deb-multimedia repository to your system by following the instructions on their site.

    2. Ensure your package manager is up to date:

    1
    $ sudo apt update && sudo apt upgrade
    1. Install Python:
    1
    $ sudo apt install python3 python3-pip python3-virtualenv
    1. Install Vapoursynth:
    1
    $ sudo apt install vapoursynth
    1. Install JET using pip:
    1
    $ pip install vsjet
    1. Ensure your package manager is up to date:
    1
    $ sudo pacman -Syu
    1. Install Python:
    1
    $ sudo pacman -S python python-pip
    1. Install Vapoursynth:
    1
    $ sudo pacman -S vapoursynth
    1. Install JET using the vapoursynth-plugin-vsjet-meta-git AUR package:
    1
    $ sudo pacman -S vapoursynth-plugin-vsjet-meta-git
    1. Install Python:
    1
    $ yay -S python python-pip python-virtualenv
    1. Install Vapoursynth:
    1
    $ sudo pacman -S vapoursynth
    1. Install JET using the vapoursynth-plugin-vsjet-meta-git AUR package:
    1
    $ yay -S vapoursynth-plugin-vsjet-meta-git
